# Changelog

All notable changes to Lotus Growth will be documented in this file.

This project follows a simple versioning rule while it is early:

- Patch releases: documentation, packaging, examples, and small CLI fixes
- Minor releases: new commands, new workflow examples, or user-facing behavior changes
- Major releases: breaking config or CLI changes

## 0.1.1 - 2026-05-10

- Added bilingual README content.
- Added `init` command for generating editable local config.
- Fixed package config loading so `pnpm dlx` can run outside the repository.
- Added screenshot, contribution guide, issue templates, CI, and solo founder SaaS example.
- Added verified temporary pnpm tarball command before npm publishing.

## 0.1.0 - 2026-05-10

- Initial open-source starter kit.
- Added minimal CLI commands: `status`, `plan`, `draft`, and `demo`.
- Added example project, agent, and account configuration.
- Added static landing page and starter documentation.
